# FontsFromWindows

常用的中文字体

字体从Windows 10当中提取。可用于Mac安装。安装步骤如下：

* 首先下载好所需的字体
* 在mac使用快捷键`command+space空格`，搜索并打开字体册
* 点击菜单`文件->添加字体`，选择所下载的字体文件，打开即可自动安装
* 安装完字体后，相关应用需要重启才能找到新安装的字体
